== Description ==
We have developed add-on for Yoast SEO plugin that enhances the current feature to next level.
So with the help of this add-on you can see the stats of SEO Titles.

= Features =
1. Very Simple Clean Code plugin to list the SEO Titles stats of yoast SEO plugin
2. Addon of yoast SEO plugin 
3. List SEO Titles by pages/ posts .
4. Colored flags: green representing SEO Titles used for single page/post and red representing SEO Titles are used for multiple pages/posts.
5. Search SEO Titles using post type and title.
6. Sort SEO Titles in ascending and descending order.
7. Pagination:10 rows per page.
8. No technical skills needed.

== Installation ==
1. Upload the folder "seo-title-stats-addon-for-yoast-seo" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to Wp-admin-> SEO Title Stats ,here you will find list of SEO Titles stats of yoast SEO plugin
